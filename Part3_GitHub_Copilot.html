GitHub Copilot 簡介與六大功能

GitHub Copilot 是什麼？
由 GitHub 和 OpenAI 合作開發，基於 Codex 模型的代碼自動補全工具。深度整合 GitHub 工作流，支援所有主流 IDE（VS Code、JetBrains、Visual Studio、Neovim）。

六大核心功能

1. 自動補全

即時預測並補全代碼，從單行到整個函數，大幅提升編寫速度

2. Copilot Chat

在 IDE 中與 AI 對話，詢問問題、解釋代碼、討論架構設計

3. 註解驅動開發

撰寫註解描述需求，Copilot 自動生成對應的完整代碼實現

4. 單元測試生成

自動為函數生成測試用例，提升代碼品質與覆蓋率

5. 程式碼重構

智能建議代碼優化方案，改善效能與可讀性

6. 文件生成

自動生成 JSDoc、docstring 等文檔註釋，保持代碼文檔同步

GitHub Copilot 安裝與授權

安裝前準備

  • GitHub 帳號（GitHub.com 免費註冊）
  • VS Code 或其他支援的 IDE
  • Copilot 訂閱（免費試用 60 天，之後 $10/月）

安裝步驟（以 VS Code 為例）

  1. 打開 VS Code 擴展市場（左側 Extensions）
  2. 搜尋 "GitHub Copilot"
  3. 點擊 "Install"，安裝官方擴展
  4. 重啟 VS Code
  5. 點擊下方狀態欄的 "Sign in with GitHub"
  6. 瀏覽器會跳轉到 GitHub 授權頁面
  7. 點擊 "Authorize"，完成授權

即時代碼建議與補全

Copilot 的兩種工作模式

自動補全

邊打邊建議，提高編寫速度 50-70%

手動觸發

按 Ctrl+Enter 手動呼喚建議

實例 1：註釋生成代碼

註釋： // 計算年齡

function calculateAge(birthDate) { const today = new Date(); const age = today.getFullYear() - birthDate.getFullYear(); const monthDiff = today.getMonth() - birthDate.getMonth(); if (monthDiff < 0 || (monthDiff === 0 && today.getDate() < birthDate.getDate())) { return age - 1; } return age; }

實例 2：測試代碼生成

已有代碼後輸入 // 測試：

const test1 = capitalize('hello'); // 'Hello' const test2 = capitalize('world'); // 'World' console.assert(test1 === 'Hello'); console.assert(test2 === 'World');

提升建議品質的方法

  • 寫明確的註釋：Copilot 會根據註釋生成代碼
  • 保留上下文：不要刪除相關代碼，讓 Copilot 理解邏輯
  • 使用類型提示：TypeScript 會產生更精準的建議
  • 循序漸進：一步步編寫，每次一個小功能

Copilot Chat 對話功能

Copilot X 的新功能
在 IDE 中開啟對話面板，像 ChatGPT 一樣與 Copilot 對話，但有代碼上下文。可以問問題、解釋代碼、生成測試、討論架構。

打開 Chat 面板

  1. VS Code 左側選擇 GitHub Copilot Chat 圖標
  2. 或按快捷鍵：Ctrl+Shift+I（Windows）/ Cmd+Shift+I（Mac）

實用的 Chat 命令

命令用途示例
/explain解釋選中代碼選中函數 → /explain
/doc生成文檔選中類 → /doc
/test生成單元測試選中函數 → /test
/fix修復代碼問題選中錯誤代碼 → /fix
@workspace引用整個專案"@workspace 怎樣組織目錄結構"

vs 自動補全的區別

  • 自動補全：快速、無腦補完代碼，適合重複性工作
  • Chat：深度對話、解釋、設計決策，適合複雜問題
建議用法
快速編寫用自動補全，遇到問題或複雜設計用 Chat
