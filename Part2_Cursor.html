Cursor IDE 簡介與特色

Cursor 是什麼？
基於 VSCode 構建的最強 AI 協作編程 IDE。完全相容 VSCode 擴展，同時深度整合多個 AI 模型（GPT-4、Claude、Gemini），提供無與倫比的代碼智能。

核心特色

Cmd/Ctrl + K

快速代碼生成與編輯，無需離開編輯器

Cmd/Ctrl + L

邊寫邊有 AI 幫你完成，像心靈感應一樣

@symbols

引用當前工作區的任何函數、類、變數

Codebase AI

AI 理解整個專案結構，實現真正的全局優化

Chat

邊修改代碼邊對話，實時反饋與修改

多模型支援

GPT-4、Claude 3.5、Gemini 隨時切換

為什麼選擇 Cursor？

  • 學習最快：無縫的 AI 體驗，幾分鐘就能上手
  • 代碼品質最高：AI 理解你的設計模式，生成的代碼風格一致
  • 社群活躍：大量教學、模板、最佳實踐共享
  • 功能持續更新：每週新功能，快速迭代

Cursor 安裝與基本設定

安裝步驟

  1. 訪問官網：https://cursor.com
  2. 下載安裝器：選擇對應系統（Windows/Mac/Linux）
  3. 執行安裝：一路 Next，約 2 分鐘完成
  4. 首次啟動：會自動引導初始設定

初始設定

選擇 AI 模型
推薦新手選擇 GPT-4 或 Claude 3.5 Sonnet。Claude 擅長中文，GPT-4 推理能力強。可在設定中隨時切換。

關鍵設定項目

設定項 建議值 說明
AI Model GPT-4 或 Claude 決定 AI 的智能程度
Temperature 0.3-0.5 越低越穩定，越高越創意
Max Tokens 2048 生成代碼的最大長度
Codebase Indexing 啟用 讓 AI 理解你的全部代碼

核心功能：AI 代碼生成

Cmd/Ctrl + K 實踐

使用場景
想要生成某段代碼時，選中代碼區域或放置光標，按 Cmd/Ctrl + K，用自然語言描述需求，Cursor 會生成完整代碼。

實例 1：生成函數

輸入提示： "寫一個函數，接收數字陣列，返回所有偶數的平方和"

function sumSquareEvenNumbers(arr) { return arr .filter(num => num % 2 === 0) .reduce((sum, num) => sum + num * num, 0); }

實例 2：生成 React 元件

輸入提示： "寫一個 React TodoList 元件，有添加、刪除、完成任務的功能"

import { useState } from 'react'; export default function TodoList() { const [todos, setTodos] = useState([]); const [input, setInput] = useState(''); const addTodo = () => { if (input.trim()) { setTodos([...todos, { id: Date.now(), text: input, done: false }]); setInput(''); } }; return ( <div className="p-6"> <h1>我的待辦清單</h1> {/* 顯示和管理任務的代碼 */} </div> ); }

進階技巧：使用 @symbols

@codebase 的魔力
在提示中輸入 @ 可快速引用已有的函數、類、變數。例如："使用 @calculateTotal 函數計算總價" 會自動引用該函數的完整定義，確保 AI 生成的代碼相容。

生成質量提升秘訣

  • 提示越詳細越好：包括輸入、輸出、邊界條件
  • 提供例子：「例如 [1,2,3] 應返回 14」會大幅提升準確度
  • 指定語言/框架：「用 React Hooks 和 Tailwind CSS」
  • 描述設計模式：「使用 MVC 架構」或「遵循 SOLID 原則」

核心功能：AI 代碼修改與重構

快速修改流程

Cmd/Ctrl + K 也用於修改
選中現有代碼，按 Cmd/Ctrl + K，輸入修改需求（如"優化性能"、"加上錯誤處理"、"轉換為異步"），Cursor 直接在原位置修改。

常見修改需求

優化性能

"優化此函數的時間複雜度"

錯誤處理

"加入 try-catch 與驗證"

單元測試

"為此函數寫 Jest 測試"

代碼重構

"轉換為函數式編程風格"

文檔化

"加上 JSDoc 註釋"

語言轉換

"將此代碼轉換為 Python"

實例：代碼優化

原始代碼：

function findDuplicates(arr) { const duplicates = []; for (let i = 0; i < arr.length; i++) { for (let j = i + 1; j < arr.length; j++) { if (arr[i] === arr[j]) { duplicates.push(arr[i]); } } } return duplicates; }

優化後（選中代碼 → Cmd/Ctrl+K → "優化性能"）：

function findDuplicates(arr) { const seen = new Set(); return [...new Set(arr.filter(item => seen.has(item) || !(seen.add(item) || false) ))]; }

批量重構提示

  • "將此檔案中的所有 var 改為 const"
  • "為所有 API 呼叫加上超時設定"
  • "將回調函數轉換為 Promise/async-await"
  • "提取魔法數字為具名常數"

實作演練：用 Cursor 建立第一個專案

目標專案：天氣查詢應用

一個簡單的 React 應用，能查詢任意城市的天氣，展示溫度、濕度、風速。

第 1 步：初始化專案

在 Cursor 終端執行： npx create-react-app weather-app cd weather-app npm install axios

第 2 步：Cursor 快速生成

  1. 打開 src/App.js
  2. 全選代碼，按 Cmd/Ctrl + K
  3. 輸入提示：
    "寫一個天氣查詢應用，有搜尋框、使用 OpenWeatherMap API、顯示溫度、濕度、風速、使用 Tailwind CSS 美化界面"
  4. 等待 AI 生成完整代碼，按 Accept

第 3 步：添加 API Key

創建 .env 檔案：

REACT_APP_API_KEY=your_openweathermap_key

第 4 步：修改與優化

修改 1：錯誤處理

選中 API 呼叫代碼 → Cmd/Ctrl+K → "加入錯誤處理與 loading 狀態"

修改 2：添加功能

新增搜尋記錄 → Cmd/Ctrl+K → "用 localStorage 保存最近 5 個搜尋"

第 5 步：運行與測試

npm start
完成！你剛剛用 Cursor 和 AI 在 10 分鐘內完成了一個完整的實用應用。傳統方式可能需要 2-3 小時。
